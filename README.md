# 自律周结算系统

一个本地可运行的自律记录小应用，用于按人、按天记录运动、读书、健康饮食、睡眠和 AI 学习表现，并按积分解锁成就与生成排行榜。

## 多人后端运行方式

启动后端服务：

```bash
node server.js
```

然后在浏览器打开 `http://localhost:3000`。多人访问同一个服务地址时，会共享同一份后端状态，数据保存在 `data/state.json`。

如果直接打开 `index.html`，页面仍可使用，但只会进入本机浏览器的本地模式，无法和其他人共享数据。

## 计分规则

- 运动：每天 12 分，周目标 4 天
- 读书：每天 8 分，周目标 5 天
- 健康饮食：每天 10 分，周目标 5 天
- 睡眠：每天 10 分，周目标 5 天
- AI 学习：每天 8 分，周目标 5 天

## 功能

- 按成员切换和新增成员
- 按周一到周日用每日卡片勾选五项成绩
- 每天显示 0/5 到 5/5 进度，三项达成显示神迹，五项全达成显示五金店
- 次日或周结算时，如某一天五项都未记录，会判定为放纵日
- 自动汇总个人本周积分、达成率和累计等级
- 每周一次性结算所有成员，同一周不能重复结算
- 总分排行榜，包含已结算积分和本周实时积分
- 个人成就墙，按里程碑、均衡、专项和积分分类展示解锁进度
- 未解锁成就会显示当前进度和目标，方便判断下一步该冲哪一枚
